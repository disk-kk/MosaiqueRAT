# MosaicRAT

### Mosaic - Remote administration tools

Mosaic is writing in C# under the "MVC" architecture, ZeroFormatter serializer is used.

![alt text](https://github.com/thdal/MosaicRAT/blob/master/Assets/MosaicRAT.png)

## Features

* UI : Flat Design 
* Tcp Protocol : IPv4 & IPv6

* Manage features :
  * [ ] Uninstall client 
  * [x] Close client

* Spying features :
  * [x] Remote desktop
  * [x] Remote shell 
  * [x] Remote webcam
  * [x] Key logger
  * [x] Password recovery

* System features :
  * [x] File manager
  * [x] Task manager
  * [x] Startup manager
  * [x] System information
  * [x] Administrator elevator
  * [ ] Registry editor
  
* Fun Features
  * [ ] Open/Close CD drive
  * [ ] Remote chat
  * [ ] Send MessageBox
  * [ ] Hide screen
  * [ ] Play song

#

## Launch Mosaic

1. Open the solution in Your IDE.
2. Run the "MosaicServeur"(Server) release.
3. Build the stub.
