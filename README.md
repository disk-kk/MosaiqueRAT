# MosaicRAT

### Mosaic - Remote administration tools

Mosaic is writing in C# under the "MVC" architecture, ZeroFormatter serializer is used.

## Features

* UI : Flat Design 
* Tcp Protocol : IPv4 & IPv6

* Spying features :
  * [x] Remote desktop
  * [x] Remote shell 
  * [x] Remote webcam
  * [x] Key logger
  * [x] Password recovery

* System features :
  * [x] File manager
  * [x] Task manager
  * [x] Startup manager
  * [x] System information
  * [x] Administrator elevator
  * [ ] Registry editor
  
* Fun Features
  * [ ] Open/Close CD drive
  * [ ] Remote chat
  * [ ] Send MessageBox
  * [ ] Hide screen
  * [ ] Play song

## Launch Mosaic

For the moment: Open the solution in Your IDE and run the "MosaicServeur"(Server) debugger you will be able to build the stub.
